Case 1.3
Responsive Web Design
https://wrd.dk/stiften/

Til case 3 skulle vi lære at skabe et website, der udforskede Mobile First paradigmet og Responsive Web Design strategien. Det krævede analyse af brugeradfæren specifikt for en mobilbruger for at udarbejde et Mobile First website. Men vi skal stadig udvikle til desktop ved hjælp af Responsive Web Design.

Vi skulle finde en rigtig kunde efter eget valg. De behøvede ikke at være indvolveret i projektet udover interviewspørgsmål.

Refleksion af
Responsive Web Design

Responsive Web Design er den tredje case som jeg har haft på denne uddannelse, den var mere "fleshed-out" i at det var et større gruppe projekt end Kulturby Aarhus 2017 og det kunne man godt mærke.

Kommunikation: Personas hjælper mig til at finde nogle trends eller mønstre, det hjælper med at se hvordan målgrupper er endten ens eller forskellige, og hvordan man kan bearbejde en hjemmeside til deres behov. Informationsarkitektur er en vigtig del fremover, alt information skal være nem at finde frem til, uden at det er til nogen besvær.

Virksomhed: Business Model Canvas og SWOT-analyser viser os hvordan jeg kan se et øjebliksbillede af virksomheder og hvordan vi kan gøre dem bedre end konkurrenter

Design: Den kreative proces følger tværfagligt med alle andre fag hånd i hånd.

Interaktion: Mobile First paradigmet og Responsive Web Design strategien er to vigtige ting som heletiden vil bruges i uddannelsen og i arbejdsforhold.
